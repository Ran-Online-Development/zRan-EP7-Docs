## Report a Bug

Use GitHub Issues in this public documentation repository for actionable bug reports.

Include:

- What happened
- What you expected to happen
- Steps to reproduce
- Client version or patch date
- Screenshots or logs with private information removed

Never post passwords, account secrets, payment details, private database data, or personal information.
